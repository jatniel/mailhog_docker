# Mailtrap Docker Image

## Usage

clone the repository: `gh repo clone jatniel/mailhog_docker`

`cd mailhog_docker && docker-compose up -d`

## view email

`localhost:8025`

## configure email
```
smpt server: localhost
smtp user: mailtrap
smtp password: mailtrap
smtp port: 1025
encryption: none
```
